### Struktur
Hikyu.js ist auf Javascript [Objekte](https://www.w3schools.com/js/js_objects.asp) aufgebaut.

```
var advData = {
    "(Ort)": {
        (Id): {
            Text: "(Dein Text)",
            Settings: {
                Typ: "Skip",
                To: (Fremd-Id)
            }
        }
    }
}
```
